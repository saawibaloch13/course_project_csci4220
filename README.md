# Hand Gesture Recognition Project
This project is designed to recognize hand gestures using computer vision and machine learning techniques. The application processes live video input to identify specific hand gestures and provides an interactive experience.

## Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Follow the steps below inn the terminal: 
- `git clone https://github.com/saawibaloch13/course_project_csci4220.git`
- `cd course_project_csci4220`
- `python3 gesture_recognition.py`
